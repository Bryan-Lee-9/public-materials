Package Managers
================

Windows (System-wide): Chocolatey_
----------------------------------

.. code:: shell

    # Admin shell
    choco search [package]
    choco install [package]

Windows (In home directory): Scoop_
-----------------------------------

Good for use on Miami's lab computers

.. code:: shell

    # User shell
    scoop search [package]
    scoop install [package]

macOS: Homebrew_
----------------

.. code:: shell

    brew

Linux: apt
----------

.. code:: shell

    apt
